BrevNine

Telegram Bot для групп • v.Brev9-24.N

BrevNine — многофункциональный помощник для Telegram-групп с ИИ-интеграцией, системой модерации и развлекательными функциями. Некоммерческий проект — всегда бесплатно!

Основные функции

  • ИИ-помощник с поддержкой изображений
  • Статистика активности участников
  • Система друзей и профилей
  • Казино с мини-играми
  • Генерация изображений
  • Скачивание музыки с SoundCloud
  • Модерация групп (в разработке)

ZetGui

ИИ-терминал & IDE • ZeroEnhanced

Революционный ИИ-агент для терминала. Забудьте про запоминание команд — просто говорите с терминалом на естественном языке. Безопасное выполнение в Docker-контейнерах.

Ключевые возможности

  • Естественный язык вместо CLI команд
  • Безопасное выполнение в Docker sandbox
  • Прозрачный ИИ с объяснением действий
  • Встроенная IDE с редактором кода
  • Атомарное редактирование файлов
  • Desktop (Neutralino) + Web версии
  • Автоматизированная установка
